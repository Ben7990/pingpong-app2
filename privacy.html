1. Общие положения

Настоящая Политика конфиденциальности описывает, как приложение «Судья Пинг-Понг PRO» (далее — «Приложение») собирает, использует и защищает информацию, которую вы предоставляете при использовании Приложения.

Использование Приложения означает ваше полное согласие с условиями настоящей Политики. Если вы не согласны с условиями, пожалуйста, не используйте Приложение.

2. Какие данные мы собираем

2.1. Данные, которые вы предоставляете добровольно:

  • Имя и фамилия — указываются при регистрации, используются для отображения в интерфейсе
  • Адрес электронной почты — используется для входа в Приложение и восстановления доступа
  • Результаты матчей и судейские протоколы — создаваемый вами контент, который вы можете экспортировать

2.2. Данные, которые мы НЕ собираем:

  • Мы НЕ собираем и НЕ храним платёжные данные (номера карт, CVC-коды, пароли от банков). Все платежи обрабатываются через сервис Boosty, который имеет собственную лицензию на обработку платежей.
  • Мы НЕ передаём ваши данные третьим лицам, за исключением случаев, прямо предусмотренных законодательством РФ.
  • Мы НЕ используем ваши данные для рекламных рассылок без вашего согласия.
  • Мы НЕ храним ваши пароли в открытом виде — используется промышленное шифрование.

3. Как мы защищаем ваши данные

Безопасность ваших данных — наш приоритет. Мы используем следующие меры защиты:

  • 🔐 Профессиональное шифрование паролей: все пароли шифруются с помощью алгоритма PBKDF2 с солью и 100 000 итерациями хеширования — это современный стандарт, используемый в банковских и государственных системах (аналогично шифрованию в онлайн-банках). Даже при получении доступа к хранилищу данных злоумышленник не сможет восстановить ваш пароль.
  • 🎲 Индивидуальная соль: для каждого пароля генерируется уникальная криптографическая соль (16 байт случайных данных), что делает невозможным использование радужных таблиц для взлома. Даже если два пользователя выберут одинаковый пароль, в хранилище они будут выглядеть по-разному.
  • 🔄 Криптостойкий алгоритм: используется Web Crypto API — встроенный в браузер криптографический модуль, который считается безопасным для финансовых операций.
  • 💻 Локальное хранение: все данные хранятся только на вашем устройстве. Мы не используем внешние серверы для хранения пользовательских данных.
  • 🔒 HTTPS: всё соединение защищено протоколом HTTPS, исключающим перехват данных при передаче.
  • 🚫 Отсутствие внешних серверов: Приложение работает полностью на вашем устройстве, данные не передаются на сторонние серверы.

4. Ваши права

В соответствии с Федеральным законом «О персональных данных» №152-ФЗ, вы имеете право:

  • На доступ к вашим данным — вы можете запросить копию всех данных, которые мы храним о вас
  • На исправление неточных данных — если какая-то информация неверна, вы можете её изменить
  • На удаление аккаунта — вы можете удалить свой аккаунт в любой момент (для этого достаточно очистить данные браузера)
  • На отзыв согласия — вы можете отозвать согласие на обработку данных

Для реализации любого из этих прав достаточно написать нам на email, указанный в разделе «Контакты».

5. Хранение данных и сроки

Ваши данные хранятся локально на вашем устройстве. При удалении аккаунта (очистке данных браузера) все ваши данные удаляются без возможности восстановления. Вы также можете очистить данные через настройки браузера. Хеши паролей хранятся бессрочно до момента удаления аккаунта.

6. Использование файлов cookie и локального хранилища

Приложение использует локальное хранилище браузера (localStorage) для сохранения:

  • Зашифрованных данных авторизации (только хеш пароля и соль, сам пароль не хранится)
  • Настроек интерфейса (выбранный язык)
  • Истории матчей и протоколов

Эти данные не передаются третьим лицам и используются только для работы Приложения. Вся чувствительная информация хранится в зашифрованном виде.

7. Платежи и пожертвования

Приложение использует Boosty для приёма добровольных пожертвований и поддержки от пользователей. Мы не обрабатываем платежи самостоятельно — вся платёжная информация обрабатывается сервисом Boosty в соответствии с их Политикой конфиденциальности.

Важно: Все пожертвования являются добровольными. PRO-функции в текущей версии Приложения предоставляются бесплатно. Пожертвования направлены на поддержку развития проекта.

8. Технические детали безопасности

Для технически подкованных пользователей:

  • Алгоритм: PBKDF2 (Password-Based Key Derivation Function 2)
  • Хеш-функция: SHA-256
  • Количество итераций: 100 000 (рекомендовано NIST)
  • Длина соли: 128 бит (16 байт криптостойкой случайности)
  • Длина выходного хеша: 256 бит (32 байта)
  • Источник случайности: crypto.getRandomValues() — криптостойкий генератор

Такая конфигурация делает перебор паролей методом "грубой силы" практически невозможным даже при наличии вычислительных кластеров.
