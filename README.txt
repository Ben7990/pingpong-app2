📁 УСТАНОВКА:
1. Создайте папку "table-tennis-scoring"
2. Сохраните файлы: index.html, styles.css, script.js
3. Откройте index.html в браузере (Chrome/Firefox/Edge)

⚡ БЫСТРЫЙ СТАРТ:
- Введите данные турнира и имена игроков
- Нажимайте кнопки "+ ОЧКО" для начисления очков
- Автоматический подсчет партий и матча

🎮 ФУНКЦИОНАЛ:

Основные кнопки:
▪️ ОЧКО ИГРОКУ 1/2 - начисление очка
▪️ НОВЫЙ МАТЧ - полный сброс
▪️ ОТМЕНА ОЧКА - отмена последнего действия

Специальные функции:
▪️ ТАЙМ-АУТ - 1 минута перерыва
▪️ ПРЕДУПРЕЖДЕНИЕ - замечание игроку
▪️ ЖЕЛТАЯ/КРАСНАЯ КАРТА - дисциплинарные санкции
▪️ СИСТЕМА УСКОРЕНИЯ - для затяжных партий
▪️ СМЕНА ПОДАЧИ/СТОРОН - изменение условий

Экспорт данных:
▪️ JSON - полный протокол для ITTF
▪️ CSV - для Excel/Google Sheets
▪️ ПЕЧАТЬ - физический протокол

📊 СТАТИСТИКА:
- Автоматический лог всех событий
- История очков (последние 5)
- Время матча в реальном времени
- Индикатор текущей подачи

🔧 ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ:
- Любой современный браузер
- JavaScript включен
- Для печати: принтер
